# 政府資料開放平臺 資料集相關 統計數據
 

### 緣由 ###

[政府資料開放平臺](http://data.gov.tw "政府資料開放平臺") 上面雖然可以下載資料集的清單，但是卻沒有 瀏覽次數、下載次數 的欄位。

我想從這兩個欄位看一下現有資料集的整體分布，因此整理了一下，放上來讓大家參考。

Note: 開放平台上的資料集更新的很快，因此在這邊所整理的數據是依據今天(2015/6/23)早上平台上所列示的資料所彙整而成的:  [政府資料開放平台 資料集 統計數據](政府資料開放平台_資料集_分析.rar "政府資料開放平台 資料集 統計數據")。

**下載 瀏覽 次數比例**

資料開放平台首頁雖然有列出"熱門"的資料集，但卻是以"瀏覽次數"為指標。我比較傾向以"下載次數"為主要指標，瀏覽而不下載 代表沒有實際使用，資料集只有被下載了才有機會發揮用途與價值。

氣象局的雷達回波資料被下載的次數與比例都是最高的。

有很多資料集的下載次數遠高於瀏覽次數，可能都是透過程式自動下載/拉取的。
